# CRUD-FASTAPI-PYTHON
Crud creado con FastApi, python y conexión a BD con mysql
1. Instalar dependencias(el archivo de requirements)
```
pip install -r requirements.txt
```
2. Configurar tu bd, llenado el .env.example con los datos de tu conexión a la BD
3. Para ejecutar el proyecto (como yo lo tengo configurado)
```
uvicorn main:app --reload
```
